**Recroogle** is a playful search engine that always returns the same result — me. No matter what you search for, the answer is always Rafael Andrade de Oliveira.

This project is a lighthearted way to showcase my profile and hopefully help me land my next opportunity. It's not meant to be disrespectful to recruiters or the hiring process — quite the opposite. It's just a fun way to say *"hey, I'm here, and I'd love to talk."*

Built with HTML, CSS, and JavaScript. Zero frameworks. Zero complexity. Just a search bar, a blue button, and one very consistent result.
